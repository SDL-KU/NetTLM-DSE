## Prerequisites

This project is recommended to run in the following environment:

* **OS:** Linux (Rocky Linux 8.10)

## Usage

To execute the design space exploration, run the following commands in your terminal:

```bash
cd NetTLM-DSE/GEMINI
sh run_dse.sh

```
